git 初始化：

git config --global core.whitespace cr-at-eol 忽略换行符的差异
git config --global core.quotepath false    让git能正常显示中文文件名
git config --global user.name "用户名"      配置用户名，全局使用该用户名登陆git
git config --global user.email "xx@qq.com"  配置邮箱，全局使用该邮箱登陆git
ls ~/.ssh/ 检查SSH KEY是否存在

2、安装包管理器（Package Control）:
到 https://packagecontrol.io/installation 安装包管理器 Package Control：
Ctrl+` （或者工具栏-视图-显示/隐藏控制台）打开控制台，输入代码，回车等待即可，
在工具栏-首选项，看到Package Control这一项，代表安装成功；
成功后所有的插件（包）会自动安装上，等待配置的完成，重启Sublime Text。
3、安装插件：
打开包管理器：ctrl + shift + p (或者工具栏-首选项-Package Control)
输入install package后回车，等待更新完成后，输入对应的插件名即可安装插件，或者下载插件安装包解压缩到Package目录(工具栏-首选项-浏览插件目录)
安装成功的插件可以在 工具栏-首选项-Package Settings 中查看和配置。
4、常用插件：
ConvertToUTF8 或者 GBK Encoding Support：	支持多种编码，解决打开文件中的中文乱码问题
Codecs33 ： 				如果不能解决全部乱码，再继续装这个插件
Emmet(原名Zen Coding)：	自动补全html/css代码
SideBarEnhancements：	改进了左侧侧边栏，增加了许多功能（复制张贴改名网页预览当前文件代码等），
Bracket Hightlighter：	用于高亮匹配括号、引号、html标签

markdown preview：		预览markdown的插件
SublimeLinter：			代码错误检查插件，高亮有错误的代码
Alignment：				代码对齐，写几个变量，选中这几行，Ctrl+Alt+A，将自动对齐

5、删除插件：
打开包管理器：ctrl + shift + p (或者工具栏-首选项-Package Control)
输入：remove package
选择：要删除的插件即可，最后插件目录对应的用户配置文件也删掉（文件路径：Sublime Text 3\Packages\User）
6、设置代码预览浏览器：
前提是装了SideBarEnhancements插件，
使用快捷键预览当前文件的代码，仅对前端html代码有效，

WampServer安装和初始化
1、安装：确定电脑是32位或64位下载相应版本，安装过程一直点击next，默认打开的浏览器可选择chrome，默认编辑器可选择sublime test；
2、安装后，启动WampServer，右下角出现W图标，如果W图标从红色变成绿色，即正常，如果是黄色，
	表示Apache或mysql没正常启动，左键点击图标看看是哪一项后重新安装；
	或者是80端口被占用了，需更改httpd.conf文件里的端口，80改为8080或其他，然后重启WampServer。
3、右键W图标，修改语言为chinese；
4、localhost/phpmyadmin 默认账户：root，密码为空
	修改密码：登陆localhost/phpmyadmin 后，账户-修改权限，可修改密码。
5、www目录（localhost指向的文件夹），想更改localhost指向的文件夹，即www目录：
	打开：D:/wamp64/script/config.inc.php，
	找到：$wwwDir = $c_installDir.'/www'，
	改为：$wwwDir = 'd:/test'，
	重启WampServer，localhost指向的文件夹就是d:/test了。
6、配置PHP的上传大小，使用内存大小：
	打开：左键W图标/php/php.ini，
	找到：upload_max_filesize = 2M（上传附件的最大值），改为喜欢的大小即可。
	找到：memory_limit = 128M（最大使用内存的大小），改为喜欢的大小即可。
7、修改phpmyadmin页面登陆的有效期
	打开：左键W图标/php/php.ini，
	找到：session.gc_maxlifetime = 1440（1440秒，即24分钟），改为喜欢的大小，接下来
	打开：D:/wamp64/apps/phpmyadmin4.6.4/libraries/config.default.php，
	找到：$cfg['LoginCookieValidity'] = 1440，修改一致即可。


myBase7 破解方法
步骤：
1、找到myBase7的安装目录(myBase.exe的目录)；
2、打开myBase.ini；
3、找到App.Lic.FirstUseOn=xxxxxx,并在确保myBase关闭的情况下将该行删除；
4、重启myBase即可正常使用；
5、因为这是重置了试用期，所以30天试用期过后还是得这样重新操作一次。
